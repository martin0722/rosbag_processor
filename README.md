# rosbag_processor

* create workspace
#### `cd rosbag_processor/src`
#### `catkin_init_workspace`

* make file executable
#### `chmod +x rosbag_processor/src/read_bag/read_bag.py`

* bulid and run
#### `cd rosbag_processor`
#### `catkin_make`
#### `source devel/setup.bash`
#### `rosrun read_bag read_bag.py [bag] [topic]`
